    # IDEA 快捷键

    快捷键 功能
    Ctrl+Alt+T 环绕
    F2/ Shift+F2 移动到有错误的代码
    Alt+Enter 快速修复
    Shift+F6 改名

    # 快捷键生成serialVersionUID

    1. 类必须实现Serializable接口

    2. （或者打开setting窗口后直接搜索 serializable class without ‘serialVersionUID’ 也可以找到这个选项） 打上 √ 选项

    3. 然后鼠标点击类型，按下快捷键alt+entry

    # 单元测试不能输入

    方法一

    1. 找到idea安装目录lib下的idea.exe.vmoptions和idea64.exe.vmoptions
    2. 在这两个文件最后加入一行 -Deditable.java.test.console=true 就ok了

    方法二

    1. Help -> Edit custom VM options
    2. 在最后一行添加：-Deditable.java.test.console=true，然后重启idea生效。
