快捷键 功能
Shift + Alt + “+” 横向分屏
Shift + Alt + “-“ 竖向分屏
Ctrl + Shift + w 取消分屏/关闭当前标签页
Ctrl + Shift + t 打开新标签页
Alt + 方向键 分屏间移动光标
Alt + Shift + 方向键 调整当前分屏区大小
Ctrl + Shift + c 复制
Ctrl + Shift + v 粘贴
Ctrl + Shift + PageUp/PageeDown 翻页
